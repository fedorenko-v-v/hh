Способ 1. Вырезать картинку и положить как background (медленная прогрузка страницы, высота блока фиксирована).
Способ 2. Вырезать 1px картинки и положить как background, размножить по горизонтали, background-repeat: repeat-x (оптимально для прогрузки страницы, высота блока фиксирована).

  3. background: -webkit-linear-gradient(top, rgba(210,255,82,1) 0%,rgba(63,198,115,1) 100%);

Блок с вертикальными вкладками

Модуль СЭД (ООО Кодекс), существующий в рабочей системе, был передан мне на переработку. Первоначально модуль имел расчеты на js. Модуль состоит из двух блоков. Первый блок - для отображения контента. Второй блок - таб с вкладками для переключения отображаемого контента ("Сообщения", "Исполнители", "Пометки") и кнопками "фиксировать блок", "свернуть-развернуть блок контента". Нижние 2 кнопки - квадратные, фиксированного размера. Табы имеют размер 150px. Активный таб должен занимать оставшееся свободное пространство.
Для того чтобы модуль адаптивно работал на экранах с разной высотой, расчет всех табов производился на js, включая перерасчет при изменении размера окна браузера. Анимация при смене активного таба выполнялась на js.
Расчет табов переложен на css благодаря flex-технологии. Перерасчет размеров всех блоков модуля при изменении окна браузера также переложен на css. Анимация смены активной вкладки сделана с помощью css анимации.
В результате процесс "свернуть-развернуть контейнер для контента" сводится к смене класса .open у блока. Анимация при смене активного таба происходит за счет установки класса .active у активного таба и переопределения его порядка в родительском элементе.
Данный пример демонстрирует, как можно снять значительную часть расчетов js и переложить их на css.

ГИДРОИЗОЛЯЦИЯ IZOMIX

Лендинг с адаптивной версткой. Адаптив сделан руками, без бутстрап. Работает на wordpress. Весь контент странички подключен к админке, можно редактировать.
В head подключены только css. Все js файлы подключены до закрытия body. Все лишние файлы, которые wordpress ставит по умолчанию, убраны, за счет этого скорость загрузки странички максимальна.
Сайт работает здесь: http://izomix.ru

Карта для "108Pixel"

Для главной страницы для сайта "108Pixel" понадобилось разработать плагин со следующими функциями:
  • Навигация по небольшой карте - виртуальной схеме студии "108Pixel";
  • Навигация с помощью курсора мыши и стрелок;
  • Приближение и отдаление схемы, глубина 1 шаг;
  • Маркеры с подписями, подписи всплывают при наведении мыши;
  • Над схемой проплывает облако, которое не должно мешать навигации и наведению на маркеры.
